# Liya Shazmina Portfolio Website

This is a static frontend website for Liya Shazmina's media/casting portfolio.

## How to open locally
Open `frontend/index.html` in a browser.

## What is included
- Profile and about section
- Video portfolio with local MP4 clips and YouTube embed
- External Instagram / YouTube watch links
- Credits section
- Photo gallery from supplied ZIP folders
- Contact section
- Download CV button

## Important updates before publishing
1. Add a domain such as `liyashazmina.com` or `liyashazminaofficial.com`.
2. Upload to Netlify, Vercel, GitHub Pages, or Azure Static Web Apps.
3. For the best casting experience, upload best videos to YouTube unlisted or Vimeo and update the links.

## Backend
No backend is required for version 1. Add a backend only if you need a contact form, admin panel, or media upload system.


Update note: This old-format version removes the embedded YouTube iframe from the video grid. The Isaiyai Nee Album remains available as a watch button under External Clips, which avoids YouTube Error 153 on local file preview.
